# Basic Order Processing

A complete e-commerce order processing workflow example.

## Overview

This example demonstrates:
- Order validation
- Payment processing
- Inventory management
- Email notifications
- Clean Architecture structure

## Source Code

View the complete source code in the [samples/basic-order-processing](https://github.com/btravers/temporal-contract/tree/main/samples/basic-order-processing) directory.

## Running the Example

```bash
cd samples/basic-order-processing
pnpm dev:worker  # Start worker
pnpm dev:client  # Run client
```

See [Examples Overview](/examples/) for more details.
